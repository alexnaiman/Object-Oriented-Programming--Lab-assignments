# Object-Oriented-Programming--Lab-assignments
Here are all my lab assingments + all the bonuses for the Object Oriented Programming course from UBB Cluj

## About this project

Each folder contains a README file with the problem statement and/or another commentaries about that project 
All the bonus points add to 1 point to the final grade

### Prerequisites

For some of the projects I used the following plugins/frameworks
 * [OpenCppCoverage Plugin](https://marketplace.visualstudio.com/items?itemName=OpenCppCoverage.OpenCppCoveragePlugin)
 * [Qt](https://www.qt.io/)
 * [JSON for Modern C++](https://github.com/nlohmann/json)
 * [MySql C++ connector](https://dev.mysql.com/downloads/connector/cpp/8.0.html)
